0. 微分方程是什么？从“变化率”讲“完整故事”

微分方程不直接描述“状态是什么”，而是描述状态如何变化。我们的任务是从“变化率”反推出“完整故事”。

  • 自变量 t：通常是时间（年、天、分钟）。
  • 因变量 y(t)：我们关心的状态（人口、温度、余额、感染人数）。
  • 右端项 f(t, y, ...)：变化的规律，由物理、生物或经济假设给出。
一般形式

dy/dt = f(t, y, parameters)

给定初值 y(t₀) 后，数值求解（如 scipy.integrate.solve_ivp）即可得到整条轨迹。

takeaway：拿到题先写“谁在变？以什么速率变？受哪些因素影响？” 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# The simplest ODE: dy/dt = 2*y
# Its solution is y(t) = y0 * exp(2*t).
def dydt(t, y):
    return 2 * y

y0 = [1.0]
t_eval = np.linspace(0, 3, 300)

# Numerical solution via solve_ivp.
sol = solve_ivp(dydt, (0, 3), y0, t_eval=t_eval)

# Compare numerical curve with the exact exponential solution.
plt.figure(figsize=(8, 4))
plt.plot(sol.t, sol.y[0], label="Numerical y(t)", linewidth=2)
plt.plot(sol.t, np.exp(2 * sol.t), "--", label="Exact e^(2t)")
plt.xlabel("Time t")
plt.ylabel("y(t)")
plt.title("An ODE describes a rate of change; solving it gives the state")
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

1a. 人口增长：指数增长 Exponential Growth

问题：小种群在资源无限时，每个个体平均产生相同数量的后代，增长率与当前人口成正比。

指数增长模型

变量：t = 时间（年），P(t) = t 时刻人口。

ODE：

dP/dt = r · P

解析解：P(t) = P₀ · e^(rt)

参数：P₀ = 1000，r = 0.03 /年。

takeaway：指数增长适合短期、资源极充裕的场景；长期必然遇到环境限制。 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# ODE: dP/dt = r * P
# The rate of change is proportional to the current population.
def exp_growth(t, P, r):
    return r * P

# Parameters and initial condition.
P0 = 1000          # initial population
r = 0.03           # per-capita growth rate (3% per year)
t_span = (0, 100)  # simulate from year 0 to year 100
t_eval = np.linspace(0, 100, 500)

# Solve numerically; args passes the extra parameter r into exp_growth.
sol = solve_ivp(exp_growth, t_span, [P0], args=(r,), t_eval=t_eval)

# Plot population versus time.
plt.figure(figsize=(8, 4))
plt.plot(sol.t, sol.y[0], label="r = {}".format(r))
plt.xlabel("Time (years)")
plt.ylabel("Population")
plt.title("Exponential Population Growth")
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

1b. 人口增长：Logistic 增长（资源限制）

问题：真实种群会碰到资源上限。加入环境容纳量 K，让增长率随 P 接近 K 而下降。

Logistic 模型

dP/dt = r·P·(1 − P/K)

含义：(1 − P/K) 是抑制因子；P 越接近 K，增长越慢。

参数：P₀ = 1000，r = 0.03，K = 10000。

takeaway：承载容量 K 把“无限增长”修正为“S 型曲线”。

启发思考：如果初始人口 P₀ > K，曲线会先下降再趋于 K；为什么？

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# ODE: dP/dt = r*P*(1 - P/K)
# Growth slows as P approaches the carrying capacity K.
def logistic_growth(t, P, r, K):
    return r * P * (1 - P / K)

# Parameters.
P0 = 1000
r = 0.03
K = 10000
t_eval = np.linspace(0, 200, 500)

# Integrate the logistic equation.
sol = solve_ivp(logistic_growth, (0, 200), [P0], args=(r, K), t_eval=t_eval)

# Plot the population curve and the carrying-capacity line.
plt.figure(figsize=(8, 4))
plt.plot(sol.t, sol.y[0], label="Logistic growth")
plt.axhline(K, color="red", linestyle="--", label="Carrying capacity K")
plt.xlabel("Time (years)")
plt.ylabel("Population")
plt.title("Logistic Population Growth")
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

2. Newton 冷却定律：一杯咖啡的温度

问题：一杯热咖啡在房间里冷却，降温速度与咖啡和环境温度的差成正比。

Newton 冷却定律

变量：t = 时间（分钟），T(t) = 咖啡温度。

ODE：

dT/dt = −k·(T − T_env)

参数：T(0)=90°C，T_env=20°C，k=0.05 /分钟。

takeaway：负号表示温度差越大、降温越快；最终 T 会趋向 T_env。

启发思考：如果咖啡杯加了保温盖（k 变小），曲线会变陡还是变缓？

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# ODE: dT/dt = -k * (T - T_env)
# Cooling rate is proportional to the temperature gap with the room.
def cooling(t, T, k, T_env):
    return -k * (T - T_env)

# Parameters.
T0 = 90            # initial coffee temperature (C)
T_env = 20         # room temperature (C)
k = 0.05           # cooling rate constant (1/min)
t_span = (0, 120)
t_eval = np.linspace(0, 120, 500)

# Solve and plot temperature versus time.
sol = solve_ivp(cooling, t_span, [T0], args=(k, T_env), t_eval=t_eval)

plt.figure(figsize=(8, 4))
plt.plot(sol.t, sol.y[0], label="Coffee temperature")
plt.axhline(T_env, color="red", linestyle="--", label="Room temperature")
plt.xlabel("Time (minutes)")
plt.ylabel("Temperature (C)")
plt.title("Newton's Law of Cooling")
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

3. 融化的雪球：几何如何简化方程

问题：球形雪球以与表面积成正比的速率融化。几何关系让这个体积问题产生出人意料的简化。

几何推导

V = (4/3)πr³，A = 4πr²，设 dV/dt = −k·A。

代入后得到：

dr/dt = −k

结论：半径以恒定速率减小，但体积按立方关系非线性下降。

takeaway：同一个物理过程，用不同变量描述会得到不同的“简单性”。

启发思考：如果雪球是立方体，融化规律还会是 dr/dt = 常数吗？

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# If dV/dt = -k * A and V = (4/3)*pi*r^3, A = 4*pi*r^2,
# the geometry collapses to dr/dt = -k.
def melting(t, r, k):
    return -k

# Parameters.
r0 = 10            # initial radius (cm)
k = 0.05           # melting speed (cm/min)
t_span = (0, 200)
t_eval = np.linspace(0, 200, 500)

# Solve for radius, then convert to volume for the second subplot.
sol = solve_ivp(melting, t_span, [r0], args=(k,), t_eval=t_eval)
r = sol.y[0]
V = (4 / 3) * np.pi * r**3

fig, axes = plt.subplots(1, 2, figsize=(12, 4))
axes[0].plot(sol.t, r, color="blue")
axes[0].set_xlabel("Time (minutes)")
axes[0].set_ylabel("Radius (cm)")
axes[0].set_title("Radius Shrinks Linearly")
axes[0].grid(True, alpha=0.3)

axes[1].plot(sol.t, V, color="cyan")
axes[1].set_xlabel("Time (minutes)")
axes[1].set_ylabel("Volume (cm^3)")
axes[1].set_title("Volume Melts Non-linearly")
axes[1].grid(True, alpha=0.3)

plt.tight_layout()
plt.show()

4. 银行账户：利息 + 连续存款

问题：银行账户余额同时受连续复利和连续存款影响。

带存款的账户模型

变量：t = 时间（年），B(t) = 余额。

ODE：

dB/dt = r·B + D

参数：B₀=1000，r=0.05，D=100（美元/年）。

takeaway：非齐次项 D 表示外部输入；即使 r=0，余额也会线性增长。

启发思考：如果每年还要扣管理费 M，方程怎么改？

import numpy as np

def bank_account(t, B, r, D):

# Parameters.

r = 0.05           # annual interest rate
D = 100            # continuous deposit rate ($/year)
t_span = (0, 30)
t_eval = np.linspace(0, 30, 500)

5. 谣言传播：社交网络的 Logistic

问题：谣言在固定人群中传播，新听到谣言的人数取决于“已知道者”与“未知道者”的接触。

谣言传播模型

变量：R(t) = 已知道人数，N = 总人口。

ODE：

dR/dt = k · R · (N − R) / N

本质：与 Logistic 方程同构，承载容量就是 N。

takeaway：“信息传播”和“种群增长”共享同一种数学结构。

启发思考：如果谣言会被人遗忘，已知道者会重新变成易感者，模型要向哪个方向扩展？

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# ODE: dR/dt = k * R * (N - R) / N
# New people hear the rumor through encounters between knowers and non-knowers.
def rumor_spread(t, R, k, N):
    return k * R * (N - R) / N

# Parameters.
N = 1000           # total population
R0 = 10            # initial knowers
k = 0.4            # spread rate (1/day)
t_span = (0, 30)
t_eval = np.linspace(0, 30, 500)

# Solve and compare against total population.
sol = solve_ivp(rumor_spread, t_span, [R0], args=(k, N), t_eval=t_eval)

plt.figure(figsize=(8, 4))
plt.plot(sol.t, sol.y[0], label="People who heard rumor")
plt.axhline(N, color="red", linestyle="--", label="Total population")
plt.xlabel("Time (days)")
plt.ylabel("R(t)")
plt.title("Spread of Rumors")
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

6. 捕食者-猎物：Lotka-Volterra 系统

问题：兔子自然增长，但被狐狸捕食；狐狸靠吃兔子繁殖，也会自然死亡。这个耦合系统产生周期性振荡。

Lotka-Volterra 方程

dR/dt = a·R − b·R·F

dF/dt = −c·F + d·R·F

参数：a=1.0, b=0.1, c=1.5, d=0.075；R(0)=40, F(0)=9。

takeaway：两个方程互相耦合，无法单独求解；数值积分是标准做法。 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# Coupled ODEs for predator-prey dynamics.
def predator_prey(t, y, a, b, c, d):
    R, F = y
    dRdt = a*R - b*R*F
    dFdt = -c*F + d*R*F
    return [dRdt, dFdt]

# Parameters and initial populations.
a, b, c, d = 1.0, 0.1, 1.5, 0.075
y0 = [40, 9]
t_span = (0, 15)
t_eval = np.linspace(0, 15, 1000)

# Integrate the coupled system.
sol = solve_ivp(predator_prey, t_span, y0, args=(a, b, c, d), t_eval=t_eval)

# Plot both populations versus time.
plt.figure(figsize=(8, 4))
plt.plot(sol.t, sol.y[0], label="Rabbits")
plt.plot(sol.t, sol.y[1], label="Foxes")
plt.xlabel("Time")
plt.ylabel("Population")
plt.title("Lotka-Volterra Predator-Prey Model")
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

6e. 拓展：给兔子加上环境容纳量 K

这部分做什么：展示如何把经典 Lotka-Volterra 模型中的“兔子无限增长”假设，改成受环境容纳量 K 限制的 Logistic 增长。

模型修改

经典模型：

dR/dt = a·R − b·R·F

引入 K 后：把兔子的自然增长项从 a·R 替换为 a·R·(1 − R/K)

dR/dt = a·R·(1 − R/K) − b·R·F

dF/dt = −c·F + d·R·F

影响：当 R 接近 K 时，兔子增长被抑制；原本闭合的相轨道变成向内螺旋，最终趋于稳定共存点。

takeaway：只需要改动兔子方程的一项，就能把“中性振荡”变成“稳定平衡”。

启发思考：K 越大，带 K 的模型越接近经典模型；如果 K 小于兔子的自然平衡值 c/d，狐狸还能存活吗？

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# Classic Lotka-Volterra: rabbits grow without bound.
def classic_lv(t, y, a, b, c, d):
    R, F = y
    dR = a*R - b*R*F
    dF = -c*F + d*R*F
    return [dR, dF]

# Modified model: rabbit growth is capped by carrying capacity K.
def logistic_lv(t, y, a, b, c, d, K):
    R, F = y
    dR = a*R*(1 - R/K) - b*R*F
    dF = -c*F + d*R*F
    return [dR, dF]

# Parameters.
a, b, c, d = 1.0, 0.1, 1.5, 0.075
K = 50
y0 = [40, 9]
t_eval = np.linspace(0, 30, 1000)

# Solve both versions.
sol_classic = solve_ivp(classic_lv, (0, 30), y0, args=(a, b, c, d), t_eval=t_eval)
sol_limited = solve_ivp(logistic_lv, (0, 30), y0, args=(a, b, c, d, K), t_eval=t_eval)

# Side-by-side comparison.
fig, axes = plt.subplots(1, 2, figsize=(14, 5))

# Time series.
axes[0].plot(sol_classic.t, sol_classic.y[0], label="Rabbits (classic)")
axes[0].plot(sol_classic.t, sol_classic.y[1], label="Foxes (classic)")
axes[0].plot(sol_limited.t, sol_limited.y[0], "--", label="Rabbits (K=50)")
axes[0].plot(sol_limited.t, sol_limited.y[1], "--", label="Foxes (K=50)")
axes[0].set_xlabel("Time")
axes[0].set_ylabel("Population")
axes[0].set_title("Classic vs. Logistic-LV Time Series")
axes[0].legend()
axes[0].grid(True, alpha=0.3)

# Phase portrait.
axes[1].plot(sol_classic.y[0], sol_classic.y[1], label="Classic (closed orbit)")
axes[1].plot(sol_limited.y[0], sol_limited.y[1], "--", label="K=50 (spiral inward)")
axes[1].set_xlabel("Rabbits")
axes[1].set_ylabel("Foxes")
axes[1].set_title("Phase Portrait Comparison")
axes[1].legend()
axes[1].grid(True, alpha=0.3)

plt.tight_layout()
plt.show()

6c. 方向场与环境容纳量

这部分做什么：在相平面上画方向场，比较“资源无限”和“资源有限（K=50）”两种假设。资源有限时，闭合轨道会变成向内螺旋。

takeaway：方向场让“跟着箭头走”变得可见；K 的引入把中性平衡变为稳定共存。 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# Shared predator-prey parameters.
a, b, c, d = 1.0, 0.1, 1.5, 0.075
K = 50  # environmental carrying capacity for rabbits

def draw_field(ax, deriv, starts, title, eq=None):
    # Grid in the (R, F) phase plane.
    R, F = np.meshgrid(np.linspace(0.1, 60, 20), np.linspace(0.1, 25, 20))
    dR, dF = deriv(R, F)
    # Normalize arrows; color by speed.
    speed = np.hypot(dR, dF)
    speed[speed == 0] = 1
    ax.quiver(R, F, dR / speed, dF / speed, speed, cmap="viridis", alpha=0.6)
    # Draw trajectories from several starting points.
    for s in starts:
        sol = solve_ivp(lambda t, y: deriv(*y), (0, 60), s,
                        t_eval=np.linspace(0, 60, 4000))
        ax.plot(sol.y[0], sol.y[1], lw=1.5)
    if eq:
        ax.plot(*eq, "ro", markersize=9)
    ax.set_xlabel("Rabbits R")
    ax.set_ylabel("Foxes F")
    ax.set_title(title)

classic = lambda R, F: (a*R - b*R*F, -c*F + d*R*F)
limited = lambda R, F: (a*R*(1 - R/K) - b*R*F, -c*F + d*R*F)

fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(13, 6))
draw_field(ax1, classic, [[20, 4], [30, 6], [40, 9]],
           "Unlimited resources: closed cycles")
draw_field(ax2, limited, [[10, 3], [40, 9], [55, 15]],
           "Carrying capacity K=50: spiral to stable point",
           eq=(c/d, a/b * (1 - (c/d)/K)))
plt.tight_layout()
plt.show()

6d. 外来物种入侵与竞争排斥

这部分做什么：让兔子-狐狸系统先达到平衡，再引入一个捕食效率更高的外来捕食者，观察竞争排斥。

takeaway：两个捕食者共享同一猎物时，所需猎物阈值更低者胜出。

启发思考：外来捕食者从未攻击狐狸，为什么狐狸仍会灭绝？

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# Three-species model: rabbits, native foxes, alien predators.
a, K = 1.0, 200
b, c, d = 0.1, 1.5, 0.075
e, g, h = 0.08, 1.2, 0.08

def system(t, y):
    R, F, A = y
    dR = a*R*(1 - R/K) - b*R*F - e*R*A
    dF = -c*F + d*R*F
    dA = -g*A + h*R*A
    return [dR, dF, dA]

# Phase 1: rabbits + foxes only.
s1 = solve_ivp(system, (0, 40), [40, 9, 0], t_eval=np.linspace(0, 40, 800))

# Phase 2: introduce 2 alien predators.
y_intro = [s1.y[0][-1], s1.y[1][-1], 2.0]
s2 = solve_ivp(system, (40, 200), y_intro, t_eval=np.linspace(40, 200, 3200))

# Concatenate phases for plotting.
t = np.concatenate([s1.t, s2.t])
R, F, A = (np.concatenate([s1.y[i], s2.y[i]]) for i in range(3))

plt.figure(figsize=(10, 5))
plt.plot(t, R, label="Rabbits (prey)")
plt.plot(t, F, label="Native foxes")
plt.plot(t, A, label="Alien predator")
plt.axvline(40, color="gray", ls="--", alpha=0.7)
plt.text(42, plt.ylim()[1]*0.9, "aliens introduced", color="gray")
plt.xlabel("Time")
plt.ylabel("Population")
plt.title("Invasion of a Foreign Predator")
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

6f. 从狼兔模型出发：10 个可探究的现实假设

建模不是“把方程算完”，而是用方程提问。下面每个假设都给出：现实意义、模型改动、以及实现提示。你可以逐个加到基础 LV 模型里，观察结果如何变化。

1. 季节性繁殖

现实：兔子在春夏繁殖更快，秋冬变慢。

改动：a(t) = a₀ + a₁·sin(2πt/T)

实现：在 ODE 里用 a0 + a1*np.sin(2*np.pi*t/T) 代替常数 a。

2. 人为捕猎狐狸

现实：保护区管理者定期捕猎狐狸控制数量。

改动：dF/dt = −c·F + d·R·F − h·F

实现：给狐狸方程加一个负的 h*F 项，h 为捕猎强度。

3. 兔子避难所

现实：部分洞穴/地形让狐狸抓不到兔子。

改动：只有暴露的兔子会被捕食：max(R − R_refuge, 0)·F

实现：把 R*F 换成 max(R - R_ref, 0)*F。

4. Holling Type II 功能反应

现实：狐狸吃兔子有“处理时间”，不能无限捕食。

改动：捕食项改为 b·R·F / (1 + e·R)

实现：定义 pred = b*R*F/(1 + e*R) 替换所有 b·R·F。

5. Allee 效应

现实：兔子太少时难以找到配偶，繁殖率反而下降。

改动：a·R 替换为 a·R·(R − A) / (K + R)

实现：设置临界值 A，当 R < A 时增长率变负。

6. 时滞响应

现实：狐狸繁殖依赖过去一段时间的兔子数量。

改动：dF/dt = −c·F + d·R(t − τ)·F

实现：用 delay differential equation 或 Euler 方法保存历史状态。

7. 兔子传染病

现实：一种疾病在兔群中传播，降低繁殖或增加死亡。

改动：把兔子拆成 S_R、I_R，加入感染与恢复/死亡。

实现：在兔子内部建立 SIR，再与狐狸捕食耦合。

8. 替代猎物

现实：狐狸在兔子少时可以吃其他小动物。

改动：给狐狸增加一个固定食物来源 s。

实现：dF/dt = −c·F + d·R·F + s，观察狐狸是否不再随兔子崩溃。

9. 环境随机扰动

现实：干旱、寒冬等随机事件影响兔子食物。

改动：a → a + σ·ξ(t)，ξ(t) 为白噪声。

实现：用 Euler-Maruyama 方法，每步加 sigma*np.sqrt(dt)*np.random.randn()。

10. 空间斑块模型

现实：不同草地的兔子会迁移，狐狸随猎物移动。

改动：多个斑块耦合：dRᵢ/dt = ... + m·(Rⱼ − Rᵢ)

实现：把状态向量扩展到所有斑块，加迁移项。

示例：把“季节性繁殖 + 狐狸捕猎”同时加入模型。这段代码展示如何在一个函数里组合多个假设，并和经典模型对比。

takeaway：每个假设只改一个或两个项；组合多个假设时，先单独验证再叠加。 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# Extended LV with seasonality and fox harvesting.
def lv_extended(t, y, a0, a1, b, c, d, h, T, K):
    R, F = y
    # Seasonal rabbit birth rate.
    a_t = a0 + a1 * np.sin(2 * np.pi * t / T)
    # Logistic growth + predation.
    dR = a_t * R * (1 - R / K) - b * R * F
    # Fox growth - natural death - harvesting.
    dF = -c * F + d * R * F - h * F
    return [dR, dF]

# Parameters.

b, c, d = 0.1, 1.5, 0.075

T = 5            # seasonal period (years)
K = 80           # carrying capacity
y0 = [40, 9]
t_eval = np.linspace(0, 30, 2000)

sol = solve_ivp(lv_extended, (0, 30), y0,
                args=(a0, a1, b, c, d, h, T, K), t_eval=t_eval)

plt.figure(figsize=(10, 4))
plt.plot(sol.t, sol.y[0], label="Rabbits")
plt.plot(sol.t, sol.y[1], label="Foxes")

7. SIR 传染病模型

问题：把人群分成易感者 S、感染者 I、康复者 R，预测疫情走势、峰值和最终规模。

SIR 模型

dS/dt = −β·S·I/N

dI/dt = β·S·I/N − γ·I

dR/dt = γ·I

基本再生数：R₀ = β/γ。

takeaway：R₀ > 1 时疫情增长；疫情结束不是因为所有人感染，而是因为易感者低于 S* = N/R₀。 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# SIR compartment model.
def sir_model(t, y, beta, gamma, N):
    S, I, R = y
    dSdt = -beta * S * I / N
    dIdt = beta * S * I / N - gamma * I
    dRdt = gamma * I
    return [dSdt, dIdt, dRdt]

# Parameters.
N = 10000
I0 = 10
S0 = N - I0
R0 = 0
beta = 0.3
gamma = 0.1
t_span = (0, 160)
t_eval = np.linspace(0, 160, 500)

# Integrate.
sol = solve_ivp(sir_model, t_span, [S0, I0, R0], args=(beta, gamma, N), t_eval=t_eval)

# Plot the three compartments.
plt.figure(figsize=(8, 4))
plt.plot(sol.t, sol.y[0], label="Susceptible")
plt.plot(sol.t, sol.y[1], label="Infectious")
plt.plot(sol.t, sol.y[2], label="Recovered")
plt.xlabel("Time (days)")
plt.ylabel("Number of people")
plt.title("SIR Model")
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

print("R0 = {:.2f}".format(beta / gamma))

7b. SIR 相平面与疫情阈值

背景：SIR 用时间序列告诉我们 S、I、R 怎么变，但相平面把“易感者 S”和“感染者 I”画在一起，让我们直接看到疫情演化的几何规律。

系数含义：β 是“一个感染者每天能传染多少易感者”（单位 1/天）；γ 是“感染者每天康复的比例”（1/天）。R₀ = β/γ 表示一个患者在完全易感人群中平均能传染多少人。

思路：令 dI/dt = 0，得到疫情转折条件 S = N/R₀。当 S 大于这个阈值时 I 上升，小于时下降，所以 I 的峰值恰好出现在 S = N/R₀ 处。

takeaway：相平面把“疫情何时见顶”转化为一个几何条件。

启发思考：R₀ 越大，阈值线越靠左；这意味着什么？

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# Compact SIR right-hand side.
def sir(t, y, beta, gamma, N):
    S, I, R = y
    return [-beta*S*I/N, beta*S*I/N - gamma*I, gamma*I]

N, I0, gamma = 10000, 10, 0.1
t_eval = np.linspace(0, 200, 1000)

# Phase-plane plot for several R0 values.
plt.figure(figsize=(8, 6))
for R0 in [0.8, 1.5, 2.5, 4.0]:

    line, = plt.plot(sol.y[0], sol.y[1], label="R0 = {:.1f}".format(R0))

7c. 免疫消退：一次性暴发还是反复波动？

背景：基础 SIR 假设康复者永久免疫，因此疫情最终消失。现实中很多疾病（如流感、新冠）康复后免疫会消退，易感者会重新补充，疫情可能反复。

系数含义：ω 是“康复者每天失去免疫、变回易感者的比例”（1/天）。ω 越大，免疫消退越快。

思路：在 R 方程中减去 ω·R，在 S 方程中加上 ω·R。长期看，系统会趋向一个稳定的地方病平衡：S* ≈ N/R₀，感染人数维持在一个非零水平。

takeaway：ω = 0 时疫情燃烧一次后消失；ω > 0 时系统回到 S* = N/R₀ 的稳态。 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# SIRS model: recovered individuals lose immunity at rate omega.
def sirs(t, y, beta, gamma, omega, N):
    S, I, R = y
    dS = -beta*S*I/N + omega*R
    dI = beta*S*I/N - gamma*I
    dR = gamma*I - omega*R
    return [dS, dI, dR]

N, I0 = 10000, 10
beta, gamma = 0.3, 0.1
t_eval = np.linspace(0, 600, 2000)

# Compare lifelong vs. fading immunity.
plt.figure(figsize=(10, 5))
for omega, label in [(0.0, "omega = 0 (lifelong)"),
                     (0.02, "omega = 0.02 (fades)")]:
    sol = solve_ivp(sirs, (0, 600), [N - I0, I0, 0], args=(beta, gamma, omega, N), t_eval=t_eval)
    plt.plot(sol.t, sol.y[1], label=label)

plt.xlabel("Time (days)")
plt.ylabel("Infectious I")
plt.title("Waning Immunity: One Outbreak vs. Recurring Waves")
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

7d. 新变种入侵与 R₀ 规则

背景：当一种新变种出现时，它和原有毒株竞争同一群易感者。谁的传播力更强，谁就能在竞争中占据优势。

系数含义：b₁、b₂ 分别是原有毒株和新变种的传播率。每个毒株的 R₀ 分别为 b₁/γ 和 b₂/γ。

思路：两种毒株都从 S 中“吸取”易感者。新毒株若 R₀ 更高，则其维持传播所需的易感者阈值 N/R₀ 更低，因此即使原有毒株还在流行，新毒株也能扩散并最终取代它。

takeaway：R₀ 更高的毒株拥有更低的易感阈值，最终取代原有毒株。

启发思考：防控时应该降低所有人的 β，还是专门针对新变种？

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# Two-strain SIR competing for the same susceptible pool.
def two_strain(t, y, b1, b2, gamma, N):
    S, I1, I2, R = y
    dS = -b1*S*I1/N - b2*S*I2/N
    dI1 = b1*S*I1/N - gamma*I1
    dI2 = b2*S*I2/N - gamma*I2
    dR = gamma*(I1 + I2)
    return [dS, dI1, dI2, dR]

N, gamma = 10000, 0.1
b1, b2 = 0.15, 0.30  # R0_1 = 1.5, R0_2 = 3.0

# Phase 1: resident strain alone.
s1 = solve_ivp(two_strain, (0, 25), [N - 10, 10, 0, 0],

# Phase 2: introduce variant.
y_intro = [s1.y[0][-1], s1.y[1][-1], 5.0, s1.y[3][-1]]
s2 = solve_ivp(two_strain, (25, 150), y_intro,
               args=(b1, b2, gamma, N), t_eval=np.linspace(25, 150, 2500))

t = np.concatenate([s1.t, s2.t])

7e. 从 SIR 出发：10 个可探究的现实假设

SIR 是最简单的传染病骨架。真实疫情中，干预措施、人口结构、病毒变异等都可以被“翻译”成方程里的新增项。下面是 10 个常见且可实现的扩展方向。

验证假设的一般方法：先明确“现实问题”→把它写成方程改动→设定参数的取值范围→运行模型→比较关键指标（峰值、最终规模、R₀、阈值）的变化。

1. 持续疫苗接种

现实：每天有一定比例的易感者被接种。

改动：dS/dt = −βSI/N − v·S，dR/dt = γI + v·S

实现：给 S 和 R 各加一项 v*S。

2. 隔离感染者

现实：出现症状后被隔离，不再传播。

改动：SIR → SIQR：dI/dt = βSI/N − γI − q·I，dQ/dt = q·I − γ_Q·Q

实现：新增 Q compartment，从 I 以速率 q 流入。

3. 无症状感染者

现实：部分感染者不发病，但也能传播。

改动：把 I 拆成 I_sym 和 I_asy，各自有传播率。

实现：dS/dt = −S·(β_sym·I_sym + β_asy·I_asy)/N。

4. 年龄结构接触

现实：儿童、成年人、老年人接触模式不同。

改动：每个年龄组一个 SIR，用接触矩阵 C_ij 描述组间传播。

实现：对每组 i：dS_i/dt = −S_i·Σ_j β·C_ij·I_j/N_j。

5. 季节性传播

现实：流感等呼吸道疾病冬季传播更强。

改动：β(t) = β₀·(1 + β₁·cos(2πt/365))

实现：在 ODE 内部用时间 t 计算 β_t。

6. 行为反馈

现实：病例增多时，人们自觉减少接触。

改动：β(I) = β₀ / (1 + k·I)

实现：让传播率随当前感染人数 I 动态下降。

7. 人口出生与死亡

现实：长期疫情中，新生儿补充易感者，死亡减少各 compartment。

改动：各 compartment 加 μ·N（出生）和 −μ·S, −μ·I, −μ·R（死亡）。

实现：把 μ 项加入每个方程，用于研究长期地方病。

8. 医疗资源饱和

现实：医院床位不足时，重症死亡率上升、恢复变慢。

改动：γ 或死亡率随 I 超过阈值后变差。

实现：γ(I) = γ₀·max(0, 1 − (I − I_cap)/I_cap) 等分段函数。

9. 疫苗突破性感染

现实：疫苗降低感染概率和重症率，但不完全阻断。

改动：新增 V compartment，V 以较低速率被感染。

实现：dV/dt = v·S − (1 − eff)·β·V·I/N。

10. 空间网络传播

现实：疾病在不同城市间通过旅行传播。

改动：每个城市一个 SIR，人群按旅行矩阵在城市间迁移。

实现：对城市 i 的 S_i 加 Σ_j m_ij·(S_j/N_j − S_i/N_i)。

示例：加入隔离 + 季节性传播。这段代码演示如何把两个假设同时写进一个函数，并与基础 SIR 对比峰值。

takeaway：干预措施通常表现为“让某些人流向新的 compartment”或“让 β 随时间/状态变化”。 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# SIR with quarantine and seasonal beta.
def sir_extended(t, y, beta0, beta1, gamma, q, gamma_q, N):
    S, I, Q, R = y
    # Seasonal transmission rate.
    beta_t = beta0 * (1 + beta1 * np.cos(2 * np.pi * t / 365))
    dS = -beta_t * S * I / N
    dI = beta_t * S * I / N - gamma * I - q * I
    dQ = q * I - gamma_q * Q
    dR = gamma * I + gamma_q * Q
    return [dS, dI, dQ, dR]

# Parameters.
N = 10000
I0 = 10
beta0, beta1 = 0.3, 0.2
gamma = 0.1
q = 0.05          # isolation rate
gamma_q = 0.14    # faster recovery in quarantine
t_span = (0, 300)
t_eval = np.linspace(0, 300, 1500)

sol = solve_ivp(sir_extended, t_span, [N - I0, I0, 0, 0],
                args=(beta0, beta1, gamma, q, gamma_q, N), t_eval=t_eval)

plt.figure(figsize=(10, 4))
plt.plot(sol.t, sol.y[1], label="Infectious I")
plt.plot(sol.t, sol.y[2], label="Quarantined Q")
plt.xlabel("Time (days)")
plt.ylabel("Number of people")
plt.title("SIR with quarantine and seasonal beta")
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

print("Peak infectious = {:.0f}".format(sol.y[1].max()))

8a. 敏感性分析：参数扫描表

背景：参数往往来自估计或文献，真实值可能偏离。敏感性分析回答：如果 β 或 γ 不准，模型预测会有多大偏差？哪种参数对结果影响最大？

系数含义：β 受接触频率、防护措施、病毒传染性影响；γ 受医疗水平、病程长短影响。公共卫生干预本质上是在降低 β 或提高 γ。

思路：一次只变一个参数（控制变量），保持其他参数不变，比较 peak_I、peak_day、final_R 和 R₀ 的变化幅度。

分析思路

固定其他参数，把 β 和 γ 分别变化 ±20%，观察 peak_I、peak_day、final_R、R₀。

takeaway：R₀ = β/γ 是决定疫情走势的核心复合参数。 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# Standard SIR model.
def sir_model(t, y, beta, gamma, N):
    S, I, R = y
    dSdt = -beta * S * I / N
    dIdt = beta * S * I / N - gamma * I
    dRdt = gamma * I
    return [dSdt, dIdt, dRdt]

N, I0 = 10000, 10
S0 = N - I0
t_span = (0, 200)
t_eval = np.linspace(0, 200, 500)

def simulate(beta, gamma):
    sol = solve_ivp(sir_model, t_span, [S0, I0, 0], args=(beta, gamma, N), t_eval=t_eval)
    I = sol.y[1]
    peak_idx = I.argmax()
    return {
        "peak_I": I.max(),
        "peak_day": sol.t[peak_idx],
        "final_R": sol.y[2][-1],
        "R0": beta / gamma
    }

# Baseline and header.
base_beta, base_gamma = 0.3, 0.1
base = simulate(base_beta, base_gamma)
print("Base: R0={:.2f}, peak={:.0f} on day {:.1f}, final R={:.0f}".format(
    base["R0"], base["peak_I"], base["peak_day"], base["final_R"]))
print("{:<18} {:<10} {:<12} {:<18} {:<18} {:<10}".format(
    "Parameter", "Change", "R0", "Peak infected", "Peak day", "Final R"))

# Vary beta and gamma by +/-20%.
for param_name, base_val in [("beta", base_beta), ("gamma", base_gamma)]:
    for pct in [-0.2, 0.0, 0.2]:
        val = base_val * (1 + pct)
        r = simulate(val, base_gamma) if param_name == "beta" else simulate(base_beta, val)
        label = "{}%".format(int(pct*100)) if pct != 0 else "base"
        print("{:<18} {:<10} {:<12.2f} {:<18.0f} {:<18.1f} {:<10.0f}".format(
            param_name, label, r["R0"], r["peak_I"], r["peak_day"], r["final_R"]))

8b. 敏感性可视化：不同 R₀ 的 I(t) 曲线

背景：参数扫描表给出具体数字，但叠加曲线能让我们一眼看到 R₀ 如何决定疫情形态。

解读：R₀ < 1 时感染单调下降；R₀ 略大于 1 时出现小峰；R₀ 很大时峰高且早。通过比较不同 R₀ 的曲线，可以直观感受干预措施（降低 β）的效果。

takeaway：R₀ 从 0.8 升到 4.0，曲线从“直接熄灭”变成“巨大峰值”。 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# Standard SIR model.
def sir_model(t, y, beta, gamma, N):
    S, I, R = y
    dSdt = -beta * S * I / N
    dIdt = beta * S * I / N - gamma * I
    dRdt = gamma * I
    return [dSdt, dIdt, dRdt]

# Fixed population and recovery rate; vary R0 by adjusting beta.
N, I0 = 10000, 10
S0 = N - I0
gamma = 0.1
t_span = (0, 200)
t_eval = np.linspace(0, 200, 500)
R0_values = [0.8, 1.2, 1.8, 2.5, 4.0]

# Plot infectious curves for a range of R0 values.
plt.figure(figsize=(10, 5))
for R0 in R0_values:
    beta = R0 * gamma
    sol = solve_ivp(sir_model, t_span, [S0, I0, 0], args=(beta, gamma, N), t_eval=t_eval)
    I = sol.y[1]
    final_size = sol.y[2][-1]
    label = "R0={:.1f}, peak={:.0f}, final={:.0f}".format(R0, I.max(), final_size)
    plt.plot(sol.t, I, label=label)

plt.axhline(0, color="gray", linestyle="--", alpha=0.5)
plt.xlabel("Time (days)")
plt.ylabel("Infectious individuals")
plt.title("SIR Infectious Curves for Different R0 Values")
plt.legend(loc="upper right")
plt.grid(True, alpha=0.3)
plt.show()

8c. 群体免疫阈值对比

背景：疫苗或自然感染让一部分人免疫，从而间接保护未免疫者。当免疫比例达到阈值时，病毒传播会被有效阻断。

系数含义：群体免疫阈值 p_c = 1 − 1/R₀。R₀ 越高，需要接种疫苗的比例就越高。

思路：把一部分初始人口直接放入 R，模拟已免疫人群，再与无免疫情景对比峰值和最终感染规模。

takeaway：当免疫比例超过阈值时，疫情规模会被显著压低。 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# Standard SIR model.
def sir_model(t, y, beta, gamma, N):
    S, I, R = y
    dSdt = -beta * S * I / N
    dIdt = beta * S * I / N - gamma * I
    dRdt = gamma * I
    return [dSdt, dIdt, dRdt]

# Disease parameters and herd-immunity threshold.
N, I0 = 10000, 10
gamma = 0.1
beta = 0.3
R0 = beta / gamma
herd_threshold = 1 - 1 / R0
vaccinated = 0.70

t_span = (0, 200)
t_eval = np.linspace(0, 200, 500)

# Scenario 1: fully susceptible.
sol_no_vacc = solve_ivp(sir_model, t_span, [N - I0, I0, 0],
                        args=(beta, gamma, N), t_eval=t_eval)

# Scenario 2: 70% initially immune.
R0_immune = int(N * vaccinated)
sol_vacc = solve_ivp(sir_model, t_span, [N - R0_immune - I0, I0, R0_immune],
                     args=(beta, gamma, N), t_eval=t_eval)

# Compare infectious curves.
plt.figure(figsize=(10, 5))
plt.plot(sol_no_vacc.t, sol_no_vacc.y[1], label="No immunity")
plt.plot(sol_vacc.t, sol_vacc.y[1], label="70% initially immune")
plt.xlabel("Time (days)")
plt.ylabel("Infectious individuals")
plt.title("Effect of Herd Immunity (R0={:.1f}, threshold={:.1%})".format(R0, herd_threshold))
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

print("Herd immunity threshold: {:.1%}".format(herd_threshold))
print("No immunity: peak = {:.0f}".format(sol_no_vacc.y[1].max()))
print("70% immune: peak = {:.0f}".format(sol_vacc.y[1].max()))

9. 数据缺失时：用模型生成合成数据

背景：真实数据可能缺失、有噪声或包含隐私问题。在模型开发阶段，用合成数据可以快速测试算法、探索极端场景、验证模型稳健性。

思路：先确定数据应服从的规律（如趋势 + 周期 + 噪声），再用随机数生成。合成数据的关键是“可控”——你知道真实参数，因此可以检验估计是否准确。

常用方法
  • 随机抽样：从分布中抽取。
  • 模型驱动：用回归或微分方程生成符合规律的数据。
  • 基于智能体：定义个体规则，让系统行为涌现。
takeaway：合成数据不是“假数据”，而是“受控实验”。 
import numpy as np
import matplotlib.pyplot as plt

# Reproducible synthetic energy-consumption data.
np.random.seed(42)
days = np.arange(1, 31)
baseline = 500
weekly = 50 * np.sin(2 * np.pi * days / 7)
noise = np.random.normal(0, 30, size=len(days))
consumption = baseline + weekly + noise

# Plot the synthetic time series.
plt.figure(figsize=(8, 4))
plt.plot(days, consumption, marker="o", label="Simulated consumption")
plt.axhline(baseline, color="red", linestyle="--", label="Baseline")
plt.xlabel("Day")
plt.ylabel("Energy (kWh)")
plt.title("Simulated Daily Energy Consumption")
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

10. 建模起手式：拿到 ODE 题先填这 6 项

问题 示例
1. 自变量是什么？时间 t（年 / 天 / 分钟）
2. 因变量是什么？人口 P、温度 T、余额 B、感染人数 I
3. 变化率由什么驱动？自身大小、与环境差距、两变量相互作用
4. 有哪些参数？增长率 r、冷却系数 k、传播率 β、恢复率 γ
5. 是否耦合？单一方程 vs. SIR / Lotka-Volterra 方程组
6. 初值条件是什么？y(0) = ?，S(0), I(0), R(0) = ?
