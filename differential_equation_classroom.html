Lecture 11 — Differential Equation Models

Classroom Notes: From Rates of Change to Full Stories

ODEs Population Dynamics Epidemics Sensitivity

A differential equation model does not describe a state directly; it describes the rate of change of that state. Our job is to take the information about "how things are changing" and piece together the full story of "what happened." Example 1 — Exponential Population Growth

Background

A small population has unlimited resources. In a short time interval, every individual is expected to produce the same number of offspring. The growth rate is therefore proportional to the current population.

Variables

  • t = time (years)
  • P(t) = population at time t

Differential Equation

dP/dt = r * P, where r is the per-capita growth rate.

Parameters (assumed)

  • Initial population: P(0) = 1000
  • Growth rate: r = 0.03 per year (3%)
  • Time horizon: 0 to 100 years
Python Demo 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

def exp_growth(t, P, r):
    return r * P

P0 = 1000
r = 0.03
t_span = (0, 100)
t_eval = np.linspace(0, 100, 500)

sol = solve_ivp(exp_growth, t_span, [P0], args=(r,), t_eval=t_eval)

plt.figure(figsize=(8, 4))
plt.plot(sol.t, sol.y[0], label="r = {}".format(r))
plt.xlabel("Time (years)")
plt.ylabel("Population")
plt.title("Exponential Population Growth")
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

Extension: Logistic Growth with Carrying Capacity

Real populations hit resource limits. Add a carrying capacity K so growth slows as P approaches K: dP/dt = r*P*(1 - P/K).

Example 2 — Newton's Law of Cooling

Background

A hot cup of coffee cools down in a room. The rate at which its temperature drops is proportional to the difference between the coffee temperature and the room temperature.

Variables

  • t = time (minutes)
  • T(t) = temperature of the coffee at time t

Differential Equation

dT/dt = -k * (T - T_env), where k > 0 controls cooling speed and T_env is the environment temperature.

Parameters (assumed)

  • Initial temperature: T(0) = 90°C
  • Room temperature: T_env = 20°C
  • Cooling constant: k = 0.05 per minute
Python Demo 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

def cooling(t, T, k, T_env):
    return -k * (T - T_env)

T0 = 90
T_env = 20
k = 0.05
t_span = (0, 120)
t_eval = np.linspace(0, 120, 500)

sol = solve_ivp(cooling, t_span, [T0], args=(k, T_env), t_eval=t_eval)

plt.figure(figsize=(8, 4))
plt.plot(sol.t, sol.y[0], label="Coffee temperature")
plt.axhline(T_env, color="red", linestyle="--", label="Room temperature")
plt.xlabel("Time (minutes)")
plt.ylabel("Temperature (C)")
plt.title("Newton's Law of Cooling")
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

Example 3 — Melting Snowball

Background

A spherical snowball melts at a rate proportional to its surface area. This sounds like a volume problem, but the geometry produces a surprising simplification.

Variables

  • t = time (minutes)
  • r(t) = radius of the snowball at time t
  • V(t) = (4/3) * π * r(t)^3
  • A(t) = 4 * π * r(t)^2

Differential Equation

dV/dt = -k * A. Substituting V and A gives dr/dt = -k. The radius shrinks at a constant rate.

Parameters (assumed)

  • Initial radius: r(0) = 10 cm
  • Melting constant: k = 0.05 cm per minute
Python Demo 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

def melting(t, r, k):
    return -k

r0 = 10
k = 0.05
t_span = (0, 200)
t_eval = np.linspace(0, 200, 500)

sol = solve_ivp(melting, t_span, [r0], args=(k,), t_eval=t_eval)
r = sol.y[0]
V = (4 / 3) * np.pi * r**3

fig, axes = plt.subplots(1, 2, figsize=(12, 4))
axes[0].plot(sol.t, r, color="blue")
axes[0].set_xlabel("Time (minutes)")
axes[0].set_ylabel("Radius (cm)")
axes[0].set_title("Radius Shrinks Linearly")
axes[0].grid(True, alpha=0.3)

axes[1].plot(sol.t, V, color="cyan")
axes[1].set_xlabel("Time (minutes)")
axes[1].set_ylabel("Volume (cm^3)")
axes[1].set_title("Volume Melts Non-linearly")
axes[1].grid(True, alpha=0.3)

plt.tight_layout()
plt.show()

Example 4 — Bank Account with Continuous Deposits

Background

Money in a savings account grows continuously due to interest, and you add money at a constant rate. The balance changes according to both effects.

Variables

  • t = time (years)
  • B(t) = account balance at time t

Differential Equation

dB/dt = r * B + D, where r is the interest rate and D is the continuous deposit rate.

Parameters (assumed)

  • Initial balance: B(0) = $1000

  • Deposit rate: D = $100 per year

Example 5 — Spread of Rumors

Background

A rumor spreads through a fixed population. The rate at which new people hear it depends on encounters between those who already know and those who do not.

Variables

  • t = time (days)
  • R(t) = number of people who have heard the rumor
  • N = total population (constant)

Differential Equation

dR/dt = k * R * (N - R) / N. This is a logistic equation with carrying capacity N.

Parameters (assumed)

  • Total population: N = 1000
  • Initial rumour carriers: R(0) = 10
  • Spread rate: k = 0.4 per day
Python Demo 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

def rumor_spread(t, R, k, N):
    return k * R * (N - R) / N

N = 1000
R0 = 10
k = 0.4
t_span = (0, 30)
t_eval = np.linspace(0, 30, 500)

sol = solve_ivp(rumor_spread, t_span, [R0], args=(k, N), t_eval=t_eval)

plt.figure(figsize=(8, 4))
plt.plot(sol.t, sol.y[0], label="People who heard rumor")
plt.axhline(N, color="red", linestyle="--", label="Total population")
plt.xlabel("Time (days)")
plt.ylabel("R(t)")
plt.title("Spread of Rumors")
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

Example 6 — Lotka-Volterra Predator-Prey

Background

A population of rabbits (prey) grows on its own but is reduced by encounters with foxes (predators). Foxes grow by eating rabbits but also die naturally. This classic coupled system produces oscillating populations.

Variables

  • t = time
  • R(t) = rabbit population
  • F(t) = fox population

Differential Equations

dR/dt = a*R - b*R*F
dF/dt = -c*F + d*R*F

Parameters (assumed)

  • Rabbit birth rate: a = 1.0
  • Predation rate: b = 0.1
  • Fox death rate: c = 1.5
  • Fox reproduction rate: d = 0.075
  • Initial rabbits: R(0) = 40; initial foxes: F(0) = 9
Python Demo 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

def predator_prey(t, y, a, b, c, d):
    R, F = y
    dRdt = a*R - b*R*F
    dFdt = -c*F + d*R*F
    return [dRdt, dFdt]

a, b, c, d = 1.0, 0.1, 1.5, 0.075
y0 = [40, 9]
t_span = (0, 15)
t_eval = np.linspace(0, 15, 1000)

sol = solve_ivp(predator_prey, t_span, y0, args=(a, b, c, d), t_eval=t_eval)

plt.figure(figsize=(8, 4))
plt.plot(sol.t, sol.y[0], label="Rabbits")
plt.plot(sol.t, sol.y[1], label="Foxes")
plt.xlabel("Time")
plt.ylabel("Population")
plt.title("Lotka-Volterra Predator-Prey Model")
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

Example 7 — SIR Epidemic Model

Background

The SIR model divides a closed population into three compartments: Susceptible, Infectious, and Recovered. It answers questions such as how many people will get sick, when the peak occurs, and whether healthcare capacity will be exceeded.

Variables

  • t = time (days)
  • S(t) = susceptible individuals
  • I(t) = infectious individuals
  • R(t) = recovered/removed individuals
  • N = S + I + R (constant closed population)

Differential Equations

dS/dt = -β * S * I / N
dI/dt = β * S * I / N - γ * I
dR/dt = γ * I

Parameters (assumed)

  • Total population: N = 10,000
  • Initial infected: I(0) = 10
  • Infection rate: β = 0.3 per day
  • Recovery rate: γ = 0.1 per day
Python Demo 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

def sir_model(t, y, beta, gamma, N):
    S, I, R = y
    dSdt = -beta * S * I / N
    dIdt = beta * S * I / N - gamma * I
    dRdt = gamma * I
    return [dSdt, dIdt, dRdt]

N = 10000
I0 = 10
S0 = N - I0
R0 = 0
beta = 0.3
gamma = 0.1
t_span = (0, 160)
t_eval = np.linspace(0, 160, 500)

sol = solve_ivp(sir_model, t_span, [S0, I0, R0], args=(beta, gamma, N), t_eval=t_eval)

plt.figure(figsize=(8, 4))
plt.plot(sol.t, sol.y[0], label="Susceptible")
plt.plot(sol.t, sol.y[1], label="Infectious")
plt.plot(sol.t, sol.y[2], label="Recovered")
plt.xlabel("Time (days)")
plt.ylabel("Number of people")
plt.title("SIR Model")
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

R_naught = beta / gamma
print("Basic reproduction number R0 = {:.2f}".format(R_naught))

Sensitivity Analysis on the SIR Model

Why It Matters

Small changes in infection rate β or recovery rate γ can flip whether an outbreak grows (R0 > 1) or dies out (R0 < 1). Varying each parameter by ±20% shows how peak infections and final outbreak size respond.

Parameters to Vary

  • Base β = 0.3, base γ = 0.1
  • Variation range: -20% and +20%
  • Output metrics: peak infectious count and total recovered by day 160
Python Demo 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

def sir_model(t, y, beta, gamma, N):
    S, I, R = y
    dSdt = -beta * S * I / N
    dIdt = beta * S * I / N - gamma * I
    dRdt = gamma * I
    return [dSdt, dIdt, dRdt]

N = 10000
I0 = 10
S0 = N - I0
t_span = (0, 160)
t_eval = np.linspace(0, 160, 500)

def simulate(beta, gamma):
    sol = solve_ivp(sir_model, t_span, [S0, I0, 0], args=(beta, gamma, N), t_eval=t_eval)
    return sol.y[1].max(), sol.y[2][-1], sol.y[1]

base_beta, base_gamma = 0.3, 0.1
base_peak, base_total, base_I = simulate(base_beta, base_gamma)

print("Base: peak infected = {:.0f}, total recovered = {:.0f}, R0 = {:.2f}".format(base_peak, base_total, base_beta/base_gamma))
print("{:<18} {:<12} {:<18} {:<18} {:<10}".format("Parameter", "Change", "Peak infected", "Total recovered", "R0"))

for param_name, base_val in [("beta", base_beta), ("gamma", base_gamma)]:
    for pct in [-0.2, 0.0, 0.2]:
        val = base_val * (1 + pct)
        if param_name == "beta":
            peak, total, _ = simulate(val, base_gamma)
            R0 = val / base_gamma
        else:
            peak, total, _ = simulate(base_beta, val)
            R0 = base_beta / val
        label = "{}%".format(int(pct*100)) if pct != 0 else "base"
        print("{:<18} {:<12} {:<18} {:<18} {:<10.2f}".format(param_name, label, "{:.0f}".format(peak), "{:.0f}".format(total), R0))

Visualization: Comparing Curves

Overlay infectious curves for the base case and the ±20% parameter changes to see how timing and height of the peak shift.

import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

def sir_model(t, y, beta, gamma, N):

    dSdt = -beta * S * I / N
    dIdt = beta * S * I / N - gamma * I

_, _, base_I = simulate(base_beta, base_gamma)

plt.figure(figsize=(8, 4))
plt.plot(t_eval, base_I, "k-", linewidth=2, label="Base beta=0.3")

for pct, color in [(-0.2, "blue"), (0.2, "red")]:

When Real Data Is Missing: Data Simulation

Why Simulate?

Real data may be unavailable, too complex, or sensitive. Synthetic data lets us test model robustness and explore "what-if" scenarios under extreme or varying conditions.

Common Approaches

  • Random sampling: Pull values from distributions (Normal, Uniform, Poisson).
  • Model-based: Use equations (regression, differential equations) to generate data that follow observed patterns.
  • Agent-based: Define rules for individual agents and let system behavior emerge.

Example: Simulate Daily Energy Consumption

Suppose a bus fleet has a baseline consumption with daily random variation. We can simulate one month of readings and add a weekly pattern.

Python Demo 
import numpy as np
import matplotlib.pyplot as plt

np.random.seed(42)
days = np.arange(1, 31)
baseline = 500
weekly = 50 * np.sin(2 * np.pi * days / 7)
noise = np.random.normal(0, 30, size=len(days))
consumption = baseline + weekly + noise

plt.figure(figsize=(8, 4))
plt.plot(days, consumption, marker="o", label="Simulated consumption")
plt.axhline(baseline, color="red", linestyle="--", label="Baseline")
plt.xlabel("Day")
plt.ylabel("Energy (kWh)")
plt.title("Simulated Daily Energy Consumption")
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()
