Lecture 11 — Differential Equation Models

Classroom Notes: From Rates of Change to Full Stories

ODEs Population Dynamics Epidemics Sensitivity

A differential equation model does not describe a state directly; it describes the rate of change of that state. Our job is to take the information about "how things are changing" and piece together the full story of "what happened." Each example below is copy-paste-ready and uses scipy.integrate.solve_ivp to numerically integrate the equations. Click the Copy button to grab any block.

ODE

Example 1 — Exponential Population Growth

Background

A small population has unlimited resources. In a short time interval, every individual is expected to produce the same number of offspring. The growth rate is therefore proportional to the current population.

Variables

  • t = time (years)
  • P(t) = population at time t

Differential Equation

dP/dt = r * P, where r is the per-capita growth rate.

Parameters (assumed)

  • Initial population: P(0) = 1000
  • Growth rate: r = 0.03 per year (3%)
  • Time horizon: 0 to 100 years
Python Demo 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

def exp_growth(t, P, r):
    return r * P

P0 = 1000
r = 0.03
t_span = (0, 100)
t_eval = np.linspace(0, 100, 500)

sol = solve_ivp(exp_growth, t_span, [P0], args=(r,), t_eval=t_eval)

plt.figure(figsize=(8, 4))
plt.plot(sol.t, sol.y[0], label="r = {}".format(r))
plt.xlabel("Time (years)")
plt.ylabel("Population")
plt.title("Exponential Population Growth")
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

Extension: Logistic Growth with Carrying Capacity

Real populations hit resource limits. Add a carrying capacity K so growth slows as P approaches K: dP/dt = r*P*(1 - P/K).

Example 2 — Newton's Law of Cooling

Background

A hot cup of coffee cools down in a room. The rate at which its temperature drops is proportional to the difference between the coffee temperature and the room temperature.

Variables

  • t = time (minutes)
  • T(t) = temperature of the coffee at time t

Differential Equation

dT/dt = -k * (T - T_env), where k > 0 controls cooling speed and T_env is the environment temperature.

Parameters (assumed)

  • Initial temperature: T(0) = 90°C
  • Room temperature: T_env = 20°C
  • Cooling constant: k = 0.05 per minute
Python Demo 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

def cooling(t, T, k, T_env):
    return -k * (T - T_env)

T0 = 90
T_env = 20
k = 0.05
t_span = (0, 120)
t_eval = np.linspace(0, 120, 500)

sol = solve_ivp(cooling, t_span, [T0], args=(k, T_env), t_eval=t_eval)

plt.figure(figsize=(8, 4))
plt.plot(sol.t, sol.y[0], label="Coffee temperature")
plt.axhline(T_env, color="red", linestyle="--", label="Room temperature")
plt.xlabel("Time (minutes)")
plt.ylabel("Temperature (C)")
plt.title("Newton's Law of Cooling")
plt.legend()
plt.grid(True, alpha=0.3)

Example 3 — Melting Snowball

Background

A spherical snowball melts at a rate proportional to its surface area. This sounds like a volume problem, but the geometry produces a surprising simplification.

Variables

  • t = time (minutes)
  • r(t) = radius of the snowball at time t
  • V(t) = (4/3) * π * r(t)^3
  • A(t) = 4 * π * r(t)^2

Differential Equation

dV/dt = -k * A. Substituting V and A gives dr/dt = -k. The radius shrinks at a constant rate.

Parameters (assumed)

  • Initial radius: r(0) = 10 cm
  • Melting constant: k = 0.05 cm per minute
Python Demo 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

def melting(t, r, k):
    return -k

r0 = 10
k = 0.05
t_span = (0, 200)
t_eval = np.linspace(0, 200, 500)

sol = solve_ivp(melting, t_span, [r0], args=(k,), t_eval=t_eval)
r = sol.y[0]
V = (4 / 3) * np.pi * r**3

fig, axes = plt.subplots(1, 2, figsize=(12, 4))
axes[0].plot(sol.t, r, color="blue")
axes[0].set_xlabel("Time (minutes)")
axes[0].set_ylabel("Radius (cm)")
axes[0].set_title("Radius Shrinks Linearly")
axes[0].grid(True, alpha=0.3)

axes[1].plot(sol.t, V, color="cyan")
axes[1].set_xlabel("Time (minutes)")
axes[1].set_ylabel("Volume (cm^3)")
axes[1].set_title("Volume Melts Non-linearly")
axes[1].grid(True, alpha=0.3)

plt.tight_layout()
plt.show()

Example 5 — Spread of Rumors

Background

A rumor spreads through a fixed population. The rate at which new people hear it depends on encounters between those who already know and those who do not.

Variables

  • t = time (days)
  • R(t) = number of people who have heard the rumor
  • N = total population (constant)

Differential Equation

dR/dt = k * R * (N - R) / N. This is a logistic equation with carrying capacity N.

Parameters (assumed)

  • Total population: N = 1000
  • Initial rumour carriers: R(0) = 10
  • Spread rate: k = 0.4 per day

Example 6 — Lotka-Volterra Predator-Prey

Background

A population of rabbits (prey) grows on its own but is reduced by encounters with foxes (predators). Foxes grow by eating rabbits but also die naturally. This classic coupled system produces oscillating populations.

Variables

  • t = time
  • R(t) = rabbit population
  • F(t) = fox population

Differential Equations

dR/dt = a*R - b*R*F
dF/dt = -c*F + d*R*F

Parameters (assumed)

  • Rabbit birth rate: a = 1.0
  • Predation rate: b = 0.1
  • Fox death rate: c = 1.5
  • Fox reproduction rate: d = 0.075
  • Initial rabbits: R(0) = 40; initial foxes: F(0) = 9
Python Demo 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

def predator_prey(t, y, a, b, c, d):
    R, F = y
    dRdt = a*R - b*R*F
    dFdt = -c*F + d*R*F
    return [dRdt, dFdt]

a, b, c, d = 1.0, 0.1, 1.5, 0.075
y0 = [40, 9]
t_span = (0, 15)
t_eval = np.linspace(0, 15, 1000)

sol = solve_ivp(predator_prey, t_span, y0, args=(a, b, c, d), t_eval=t_eval)

plt.figure(figsize=(8, 4))
plt.plot(sol.t, sol.y[0], label="Rabbits")
plt.plot(sol.t, sol.y[1], label="Foxes")
plt.xlabel("Time")
plt.ylabel("Population")
plt.title("Lotka-Volterra Predator-Prey Model")
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

Extension: Direction Field & Environmental Limits

A direction field draws the arrow (dR/dt, dF/dt) at each point, so any trajectory simply follows the arrows. The classic model has no resource limit on rabbits, so every orbit is a closed cycle circling the equilibrium forever. But if the environment can only sustain so many rabbits, we add a carrying capacity K to the prey equation: dR/dt = a*R*(1 - R/K) - b*R*F. Watch how the same arrows now spiral inward to a single stable point instead of looping.

Python Demo 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

a, b, c, d = 1.0, 0.1, 1.5, 0.075
K = 50  # environmental carrying capacity for rabbits

def draw_field(ax, deriv, starts, title, eq=None):
    R, F = np.meshgrid(np.linspace(0.1, 60, 20), np.linspace(0.1, 25, 20))
    dR, dF = deriv(R, F)
    speed = np.hypot(dR, dF)
    speed[speed == 0] = 1
    ax.quiver(R, F, dR / speed, dF / speed, speed, cmap="viridis", alpha=0.6)
    for s in starts:
        sol = solve_ivp(lambda t, y: deriv(*y), (0, 60), s,
                        t_eval=np.linspace(0, 60, 4000))
        ax.plot(sol.y[0], sol.y[1], lw=1.5)
    if eq:
        ax.plot(*eq, "ro", markersize=9)
    ax.set_xlabel("Rabbits R")
    ax.set_ylabel("Foxes F")
    ax.set_title(title)

classic = lambda R, F: (a*R - b*R*F, -c*F + d*R*F)
limited = lambda R, F: (a*R*(1 - R/K) - b*R*F, -c*F + d*R*F)

fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(13, 6))
draw_field(ax1, classic, [[20, 4], [30, 6], [40, 9]],
           "Unlimited resources: closed cycles (neutral)")
draw_field(ax2, limited, [[10, 3], [40, 9], [55, 15]],
           "Carrying capacity K=50: spiral to stable coexistence",
           eq=(c/d, a/b * (1 - (c/d)/K)))
plt.tight_layout()
plt.show()

print("Equilibrium rabbits R* = c/d = {:.0f} (same in both models)".format(c/d))
print("Classic foxes  F* = a/b = {:.0f}  -> orbits circle this point forever".format(a/b))
print("Limited foxes  F* = {:.1f}     -> trajectories settle here".format(a/b * (1 - (c/d)/K)))

Think about it: The classic model assumes rabbits could multiply without bound if foxes vanished — an environment with infinite production. That is why its orbits never settle: nothing damps the swings. Adding K says the land can only feed so many rabbits. Now the arrows pull every path toward one coexistence point, and the boom-and-bust cycles fade into a steady balance. Try shrinking K toward R* = c/d = 20: what happens to the foxes when the environment can barely support the prey they depend on? Which is more realistic for a real ecosystem — endless cycles, or a resource-limited balance that absorbs shocks?

Extension: Invasion of a Foreign Species

Now let the rabbit-fox system settle near its equilibrium, then release an alien predator A that hunts the very same rabbits. The community now has two predators feeding on one prey. To model it we keep the native equations and add a third: the alien grows only when it catches enough rabbits, dA/dt = -g*A + h*R*A, while its hunting adds a new drain -e*R*A to the rabbits.

Extended Equations

dR/dt = a*R*(1 - R/K) - b*R*F - e*R*A
dF/dt = -c*F + d*R*F
dA/dt = -g*A + h*R*A

Analysing the Alien

A predator can only survive if the prey stays above its personal break-even level: foxes need R >= c/d, the alien needs R >= g/h. Whichever predator can persist on less prey wins — it grazes the rabbits down past its rival's survival line, starving it out. This is the ecological "R* rule" of competition. We deliberately give the alien a lower threshold (g/h = 15 < c/d = 20), so watch the clash unfold.

Python Demo 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

a, K = 1.0, 200            # rabbit growth, carrying capacity
b, c, d = 0.1, 1.5, 0.075  # native fox: predation, death, reproduction
e, g, h = 0.08, 1.2, 0.08  # alien:      predation, death, reproduction

def system(t, y):
    R, F, A = y
    dR = a*R*(1 - R/K) - b*R*F - e*R*A
    dF = -c*F + d*R*F
    dA = -g*A + h*R*A
    return [dR, dF, dA]

# Phase 1: rabbits + foxes only, allowed to reach equilibrium
s1 = solve_ivp(system, (0, 40), [40, 9, 0], t_eval=np.linspace(0, 40, 800))

# Phase 2: drop in 2 alien predators and continue
y_intro = [s1.y[0][-1], s1.y[1][-1], 2.0]
s2 = solve_ivp(system, (40, 200), y_intro, t_eval=np.linspace(40, 200, 3200))

t = np.concatenate([s1.t, s2.t])
R, F, A = (np.concatenate([s1.y[i], s2.y[i]]) for i in range(3))

plt.figure(figsize=(10, 5))
plt.plot(t, R, label="Rabbits (prey)")
plt.plot(t, F, label="Native foxes")
plt.plot(t, A, label="Alien predator")
plt.axvline(40, color="gray", ls="--", alpha=0.7)
plt.text(42, plt.ylim()[1]*0.9, "aliens introduced", color="gray")
plt.xlabel("Time")
plt.ylabel("Population")
plt.title("Invasion of a Foreign Predator Sharing the Same Prey")
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

print("Fox   survives only if rabbits >= c/d = {:.1f}".format(c/d))
print("Alien survives only if rabbits >= g/h = {:.1f}".format(g/h))
print("Final: rabbits={:.1f}, foxes={:.3f}, aliens={:.1f}".format(R[-1], F[-1], A[-1]))

Think about it: The foxes did nothing wrong — their birth, death, and hunting rates never changed. Yet a single introduction wiped them out, because the alien could survive on fewer rabbits and pulled the prey below the foxes' survival line. Extinction here is competitive, not a direct attack. Ask yourself: the alien never touches a fox, so why does the fox vanish? What single number decided the winner, and what would you have to change about the fox (c, d) or the environment (K) to let both predators coexist? And if you were managing this ecosystem, would you regulate the invader's hunting efficiency (h), boost the prey's carrying capacity (K), or act before the equilibrium ever clashed?

Sensitivity

Sensitivity Analysis on the Predator-Prey Model

Why It Matters

Small changes in birth, death, or predation rates can alter cycle amplitude, average population levels, and even whether both species persist. Varying each parameter by ±20% reveals which rates most strongly control the ecosystem.

Parameters to Vary

  • Base: a = 1.0, b = 0.1, c = 1.5, d = 0.075
  • Variation range: -20% and +20%
  • Output metrics: peak rabbits, peak foxes, average rabbits, average foxes
Python Demo 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

def predator_prey(t, y, a, b, c, d):
    R, F = y
    dRdt = a*R - b*R*F
    dFdt = -c*F + d*R*F
    return [dRdt, dFdt]

def simulate(a=1.0, b=0.1, c=1.5, d=0.075, y0=(40, 9)):
    t_eval = np.linspace(0, 30, 1000)
    sol = solve_ivp(predator_prey, (0, 30), y0, args=(a, b, c, d), t_eval=t_eval)
    return sol

def metrics(sol):
    R, F = sol.y
    return {
        "R_peak": R.max(),
        "F_peak": F.max(),
        "R_mean": R.mean(),
        "F_mean": F.mean()
    }

base = simulate()
base_m = metrics(base)
print("Base: a=1.0, b=0.1, c=1.5, d=0.075")
print("R_peak={:.1f}, F_peak={:.1f}, R_mean={:.1f}, F_mean={:.1f}".format(
    base_m["R_peak"], base_m["F_peak"], base_m["R_mean"], base_m["F_mean"]))

params = {
    "Rabbit birth a": ("a", 1.0),
    "Predation b": ("b", 0.1),
    "Fox death c": ("c", 1.5),
    "Fox repro d": ("d", 0.075)
}

print("{:<18} {:<10} {:<10} {:<10} {:<10}".format(
    "Parameter", "R_peak", "F_peak", "R_mean", "F_mean"))
for name, (key, val) in params.items():
    for pct in [-0.2, 0.2]:
        kwargs = {"a": 1.0, "b": 0.1, "c": 1.5, "d": 0.075}
        kwargs[key] = val * (1 + pct)
        sol = simulate(**kwargs)
        m = metrics(sol)
        label = "{} {}".format(name, "+20%" if pct > 0 else "-20%")
        print("{:<18} {:<10.1f} {:<10.1f} {:<10.1f} {:<10.1f}".format(
            label, m["R_peak"], m["F_peak"], m["R_mean"], m["F_mean"]))

Epidemic

Example 7 — SIR Epidemic Model

Background

The SIR model divides a closed population into three compartments: Susceptible, Infectious, and Recovered. It answers questions such as how many people will get sick, when the peak occurs, and whether healthcare capacity will be exceeded.

Variables

  • t = time (days)
  • S(t) = susceptible individuals
  • I(t) = infectious individuals
  • R(t) = recovered/removed individuals
  • N = S + I + R (constant closed population)

Differential Equations

dS/dt = -β * S * I / N
dI/dt = β * S * I / N - γ * I
dR/dt = γ * I

Parameters (assumed)

  • Total population: N = 10,000
  • Initial infected: I(0) = 10
  • Infection rate: β = 0.3 per day
  • Recovery rate: γ = 0.1 per day
Python Demo 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

def sir_model(t, y, beta, gamma, N):
    S, I, R = y
    dSdt = -beta * S * I / N
    dIdt = beta * S * I / N - gamma * I
    dRdt = gamma * I
    return [dSdt, dIdt, dRdt]

N = 10000
I0 = 10
S0 = N - I0
R0 = 0
beta = 0.3
gamma = 0.1
t_span = (0, 160)
t_eval = np.linspace(0, 160, 500)

sol = solve_ivp(sir_model, t_span, [S0, I0, R0], args=(beta, gamma, N), t_eval=t_eval)

plt.figure(figsize=(8, 4))
plt.plot(sol.t, sol.y[0], label="Susceptible")
plt.plot(sol.t, sol.y[1], label="Infectious")
plt.plot(sol.t, sol.y[2], label="Recovered")
plt.xlabel("Time (days)")
plt.ylabel("Number of people")
plt.title("SIR Model")
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

R_naught = beta / gamma
print("Basic reproduction number R0 = {:.2f}".format(R_naught))
Sensitivity

Sensitivity Analysis on the SIR Model

Why It Matters

The basic reproduction number R0 = β/γ determines almost everything: whether an outbreak grows, how high the peak is, when it occurs, and how many people are ultimately infected. Changing β and γ by ±20% can flip R0 across the critical threshold of 1.

Part 1: ±20% Parameter Sweep

Vary β and γ individually by ±20% while holding the other fixed. Report peak infections, peak day, final recovered count, and R0.

Python Demo 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

def sir_model(t, y, beta, gamma, N):
    S, I, R = y
    dSdt = -beta * S * I / N
    dIdt = beta * S * I / N - gamma * I
    dRdt = gamma * I
    return [dSdt, dIdt, dRdt]

N = 10000
I0 = 10
S0 = N - I0
t_span = (0, 200)
t_eval = np.linspace(0, 200, 500)

def simulate(beta, gamma):
    sol = solve_ivp(sir_model, t_span, [S0, I0, 0], args=(beta, gamma, N), t_eval=t_eval)
    I = sol.y[1]
    peak_idx = I.argmax()
    return {
        "peak_I": I.max(),
        "peak_day": sol.t[peak_idx],
        "final_R": sol.y[2][-1],
        "R0": beta / gamma
    }

base_beta, base_gamma = 0.3, 0.1
base = simulate(base_beta, base_gamma)
print("Base: R0={:.2f}, peak={:.0f} on day {:.1f}, final recovered={:.0f}".format(
    base["R0"], base["peak_I"], base["peak_day"], base["final_R"]))
print("{:<18} {:<10} {:<12} {:<18} {:<18} {:<10}".format(
    "Parameter", "Change", "R0", "Peak infected", "Peak day", "Final R"))

for param_name, base_val in [("beta", base_beta), ("gamma", base_gamma)]:
    for pct in [-0.2, 0.0, 0.2]:
        val = base_val * (1 + pct)
        if param_name == "beta":
            r = simulate(val, base_gamma)
        else:
            r = simulate(base_beta, val)
        label = "{}%".format(int(pct*100)) if pct != 0 else "base"
        print("{:<18} {:<10} {:<12.2f} {:<18.0f} {:<18.1f} {:<10.0f}".format(
            param_name, label, r["R0"], r["peak_I"], r["peak_day"], r["final_R"]))

Part 2: R0 Scenarios

Fix γ = 0.1 and choose β to obtain R0 values spanning below, near, and above 1. This directly shows the threshold behavior and how small changes in R0 translate into large differences in outbreak size.

Part 3: Herd Immunity Threshold

For R0 = 3.0, the herd immunity threshold is 1 - 1/R0 ≈ 66.7%. Simulate what happens if 70% of the population is initially immune (e.g., vaccinated) compared with a fully susceptible population.

Python Demo 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

def sir_model(t, y, beta, gamma, N):

N = 10000
I0 = 10
gamma = 0.1
beta = 0.3
R0 = beta / gamma
herd_threshold = 1 - 1 / R0
vaccinated = 0.70

sol_no_vacc = solve_ivp(sir_model, t_span, [N - I0, I0, 0],

R0_immune = int(N * vaccinated)
sol_vacc = solve_ivp(sir_model, t_span, [N - R0_immune - I0, I0, R0_immune],

plt.figure(figsize=(10, 5))
plt.plot(sol_no_vacc.t, sol_no_vacc.y[1], label="No immunity")
plt.plot(sol_vacc.t, sol_vacc.y[1], label="70% initially immune")

plt.ylabel("Infectious individuals")
plt.title("Effect of Herd Immunity (R0={:.1f}, threshold={:.1%})".format(R0, herd_threshold))

plt.show()

When Real Data Is Missing: Data Simulation

Why Simulate?

Real data may be unavailable, too complex, or sensitive. Synthetic data lets us test model robustness and explore "what-if" scenarios under extreme or varying conditions.

Common Approaches

  • Random sampling: Pull values from distributions (Normal, Uniform, Poisson).
  • Model-based: Use equations (regression, differential equations) to generate data that follow observed patterns.
  • Agent-based: Define rules for individual agents and let system behavior emerge.

Example: Simulate Daily Energy Consumption

Suppose a bus fleet has a baseline consumption with daily random variation. We can simulate one month of readings and add a weekly pattern.
