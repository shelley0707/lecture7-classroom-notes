1. 概率分布是随机性的积木

在建模前，先要判断随机变量适合用什么分布描述。PPT 中提到的核心分布如下：

分布适用场景Python
Normal身高、误差、任务耗时np.random.normal(μ, σ)
Binomialn 次独立成功次数np.random.binomial(n, p)
Poisson单位时间事件发生次数np.random.poisson(λ)
Uniform等可能取值np.random.uniform(a, b)
Exponential事件间隔时间 / 寿命np.random.exponential(1/λ)
takeaway：选分布 = 判断变量是连续/离散、是否有上界、事件是否独立。 
import numpy as np
import matplotlib.pyplot as plt

np.random.seed(42)
n_samples = 10000

fig, axes = plt.subplots(2, 3, figsize=(14, 8))
samples = {
    'Normal(5, 3)': np.random.normal(5, 3, n_samples),
    'Binomial(10, 0.5)': np.random.binomial(10, 0.5, n_samples),
    'Poisson(3)': np.random.poisson(3, n_samples),
    'Uniform(0, 1)': np.random.uniform(0, 1, n_samples),
    'Exponential(λ=0.5)': np.random.exponential(1/0.5, n_samples)
}
for ax, (name, data) in zip(axes.flat, samples.items()):
    ax.hist(data, bins=30, color='steelblue', edgecolor='white', alpha=0.8)
    ax.set_title(name)
    ax.set_xlabel('Value')
    ax.set_ylabel('Frequency')
axes.flat[-1].axis('off')
plt.suptitle('Common Probability Distributions')
plt.tight_layout()
plt.show()

2. Monte Carlo 案例：飘落的叶子

场景：一片叶子从 5 米高的树枝脱落。水平风速 V ~ Normal(μ=5, σ=3) m/s，垂直下落速度恒为 1 m/s。求叶子落地点的分布。

模型设定

已知参数：

  • h = 5 m：初始高度
  • vvertical = 1 m/s：垂直下落速度
  • V ~ N(5, 3)：水平风速

核心变量：

  • T = h / vvertical = 5 s：下落时间
  • X = V · T：水平落点距离

分布结论：

X ~ N(μ·T, σ·T) = N(25, 15)

takeaway：一个简单的随机输入（风速）乘以一个确定的时间，就得到随机输出（落点）。 
import numpy as np
import matplotlib.pyplot as plt

np.random.seed(42)
n = 10000
h = 5.0          # m
v_vert = 1.0     # m/s
mu_v = 5.0       # m/s
sigma_v = 3.0    # m/s

T = h / v_vert                    # 下落时间
V = np.random.normal(mu_v, sigma_v, n)
X = V * T

print(f'Fall time: {T:.1f} s')
print(f'Mean landing distance: {X.mean():.2f} m')
print(f'Std landing distance: {X.std():.2f} m')
print(f'P(landing between 10m and 40m): {((X >= 10) & (X <= 40)).mean():.2%}')

# 可视化
fig, axes = plt.subplots(1, 2, figsize=(13, 4))
axes[0].hist(V, bins=40, color='skyblue', edgecolor='white')
axes[0].set_title('Wind Speed Distribution V ~ N(5, 3)')
axes[0].set_xlabel('Wind speed (m/s)')
axes[0].set_ylabel('Count')

axes[1].hist(X, bins=40, color='seagreen', edgecolor='white')
axes[1].axvline(X.mean(), color='red', linestyle='--', label=f'Mean={X.mean():.1f}m')
axes[1].set_title('Landing Distance Distribution X = V · T')
axes[1].set_xlabel('Horizontal distance (m)')
axes[1].set_ylabel('Count')
axes[1].legend()
plt.tight_layout()
plt.show()

3. Monte Carlo 案例：估算 π

思想：向边长为 2 的正方形内随机撒点，其中落在内切圆内的比例 ≈ 圆面积 / 正方形面积 = π/4。

数学表达

随机变量：(X, Y) ~ Uniform(-1, 1) × Uniform(-1, 1)

指示变量：

I = 1 if X² + Y² ≤ 1, else 0

估计量：

π̂ = 4 · (ΣI) / N

收敛：当 N → ∞，π̂ → π（大数定律）。

takeaway：用随机实验估算确定值，是 Monte Carlo 最直观的演示。 
import numpy as np
import matplotlib.pyplot as plt

np.random.seed(42)
N = 5000
x = np.random.uniform(-1, 1, N)
y = np.random.uniform(-1, 1, N)
inside = x**2 + y**2 <= 1

pi_est = 4 * inside.sum() / N
print(f'Estimated π with N={N}: {pi_est:.5f}')
print(f'Error: {abs(np.pi - pi_est):.5f}')

# 可视化撒点
fig, axes = plt.subplots(1, 2, figsize=(12, 5))
axes[0].scatter(x[inside], y[inside], c='blue', s=5, alpha=0.5, label='Inside circle')
axes[0].scatter(x[~inside], y[~inside], c='orange', s=5, alpha=0.5, label='Outside circle')
theta = np.linspace(0, 2*np.pi, 100)
axes[0].plot(np.cos(theta), np.sin(theta), 'r-', linewidth=2)
axes[0].set_aspect('equal')
axes[0].set_title(f'Estimate π ≈ {pi_est:.4f}')
axes[0].legend()

# 收敛曲线
Ns = np.logspace(2, 5, 50, dtype=int)
ests = []
for n in Ns:
    xx = np.random.uniform(-1, 1, n)
    yy = np.random.uniform(-1, 1, n)
    ests.append(4 * (xx**2 + yy**2 <= 1).sum() / n)
axes[1].plot(Ns, ests, label='MC estimate')
axes[1].axhline(np.pi, color='red', linestyle='--', label='True π')
axes[1].set_xscale('log')
axes[1].set_title('Convergence of π Estimate')
axes[1].set_xlabel('Number of samples')
axes[1].set_ylabel('π estimate')
axes[1].legend()
plt.tight_layout()
plt.show()

4. Monte Carlo 案例：产品上线时间风险

场景：一个科技初创公司准备发布旗舰产品。各任务持续时间是随机变量，需要估计总完成时间、30 天内上线概率、关键风险点。

任务与依赖
  • A: Feature Dev — N(10, 2)
  • B: UI/UX — N(7, 1.5)
  • C: Backend — N(8, 2)，依赖 A
  • D: Frontend — N(6, 1)，依赖 B
  • E: Testing — U(5, 10)，依赖 C, D
  • F: Marketing — N(12, 3)
  • G: Legal — U(3, 7)，依赖 A, B
  • H: Launch — 1 天，依赖 E, F, G

关键路径：

T = max(A+C+E, B+D+E, A+G, B+G, F) + H

takeaway：Monte Carlo 能直接给出总工期的分布，而不是一个“最可能日期”。 
import numpy as np
import matplotlib.pyplot as plt

np.random.seed(42)
n = 10000

# 每次试验抽样各任务时长
A = np.random.normal(10, 2, n)
B = np.random.normal(7, 1.5, n)
C = np.random.normal(8, 2, n)
D = np.random.normal(6, 1, n)
E = np.random.uniform(5, 10, n)
F = np.random.normal(12, 3, n)
G = np.random.uniform(3, 7, n)
H = 1

# 计算关键路径
path1 = A + C + E
path2 = B + D + E
path3 = A + G
path4 = B + G
path5 = F

T = np.maximum.reduce([path1, path2, path3, path4, path5]) + H

print(f'Mean completion time: {T.mean():.1f} days')
print(f'P(finish within 30 days): {(T <= 30).mean():.2%}')
print(f'P(finish within 35 days): {(T <= 35).mean():.2%}')
print(f'95% confidence interval: [{np.percentile(T, 2.5):.1f}, {np.percentile(T, 97.5):.1f}]')

# 可视化
fig, axes = plt.subplots(1, 2, figsize=(13, 4))
axes[0].hist(T, bins=50, color='coral', edgecolor='white')
axes[0].axvline(30, color='red', linestyle='--', label='30-day deadline')
axes[0].set_title('Distribution of Total Completion Time')
axes[0].set_xlabel('Days')
axes[0].set_ylabel('Frequency')
axes[0].legend()

# Tornado: 每条路径对总工期的贡献（用相关系数近似）
paths = {'A+C+E': path1, 'B+D+E': path2, 'A+G': path3, 'B+G': path4, 'F': path5}
corrs = {k: np.corrcoef(v, T)[0, 1] for k, v in paths.items()}
axes[1].barh(list(corrs.keys()), list(corrs.values()), color='steelblue')
axes[1].set_title('Path Correlation with Total Time (Risk Ranking)')
axes[1].set_xlabel('Correlation')
plt.tight_layout()
plt.show()

5. Agent-Based Simulation：狼-羊-草生态系统

核心思想：从个体（agent）的简单规则出发，观察群体层面的涌现行为。三个要素：Agent、Environment、Rules。

模型规则

羊 Sheep：

  • 每步随机移动，能量下降
  • 吃到草 → 能量增加
  • 能量 > 阈值 → 繁殖（分裂）
  • 能量 ≤ 0 → 死亡

狼 Wolves：

  • 每步随机移动，能量下降
  • 遇到羊 → 吃掉，能量大增
  • 能量 > 阈值 → 繁殖
  • 能量 ≤ 0 → 死亡

草 Grass：

  • 被吃掉后按一定速率重新生长
takeaway：简单局部规则 → 复杂全局模式（捕食者-猎物周期振荡）。 
import numpy as np
import matplotlib.pyplot as plt

np.random.seed(42)

# 简化版狼-羊-草（Lotka-Volterra 随机化近似）
n_steps = 200
sheep = np.zeros(n_steps)
wolves = np.zeros(n_steps)
sheep[0], wolves[0] = 40, 9

# 参数
alpha = 0.1    # 羊自然增长率
beta = 0.02    # 狼吃羊效率
gamma = 0.1    # 狼自然死亡率
delta = 0.01   # 狼吃羊后增长

for t in range(n_steps - 1):
    # 加入随机噪声模拟 ABS 的离散事件
    noise_s = np.random.normal(0, 1)
    noise_w = np.random.normal(0, 0.5)
    ds = alpha * sheep[t] - beta * sheep[t] * wolves[t] + noise_s
    dw = delta * sheep[t] * wolves[t] - gamma * wolves[t] + noise_w
    sheep[t+1] = max(0, sheep[t] + ds)
    wolves[t+1] = max(0, wolves[t] + dw)

# 可视化
fig, axes = plt.subplots(1, 2, figsize=(13, 4))
axes[0].plot(sheep, label='Sheep', color='green')
axes[0].plot(wolves, label='Wolves', color='red')
axes[0].set_title('Wolf-Sheep Population Dynamics')
axes[0].set_xlabel('Time step')
axes[0].set_ylabel('Population')
axes[0].legend()

axes[1].plot(sheep, wolves, color='purple', alpha=0.7)
axes[1].scatter(sheep[0], wolves[0], color='green', s=100, zorder=5, label='Start')
axes[1].scatter(sheep[-1], wolves[-1], color='red', s=100, zorder=5, label='End')
axes[1].set_title('Phase Space: Sheep vs Wolves')
axes[1].set_xlabel('Sheep')
axes[1].set_ylabel('Wolves')
axes[1].legend()
plt.tight_layout()
plt.show()

6. Markov Chain 案例：共享单车分布

场景：三个共享单车站点 A、B、C。用户骑车的还车偏好稳定，但自行车在空间分布上不断变化。预测长期分布。

Markov 模型

状态空间：S = {A, B, C}

状态分布向量：πt = [πA(t), πB(t), πC(t)]

转移矩阵 P：

Pij = P(Xt+1 = j | Xt = i)

迭代公式：

πt+1 = πt · P

稳态分布：

π = πP

takeaway：Markov 的核心是“无记忆性”：下一状态只取决于当前状态。 
import numpy as np
import matplotlib.pyplot as plt

# 转移矩阵：行 = 当前站，列 = 下一站
# 示例：从 A 出发，60% 还 A，30% 还 B，10% 还 C
P = np.array([
    [0.6, 0.3, 0.1],
    [0.2, 0.5, 0.3],
    [0.25, 0.25, 0.5]
])

# 初始分布：所有车都在 A
pi = np.array([1.0, 0.0, 0.0])

steps = 30
history = [pi]
for _ in range(steps):
    pi = pi @ P
    history.append(pi)
history = np.array(history)

print('Distribution after 30 steps:', history[-1])

# 求稳态分布（特征值为 1 的左特征向量）
eigenvalues, eigenvectors = np.linalg.eig(P.T)
idx = np.argmin(np.abs(eigenvalues - 1))
stationary = eigenvectors[:, idx].real
stationary = stationary / stationary.sum()
print('Stationary distribution:', stationary)

# 可视化
fig, axes = plt.subplots(1, 2, figsize=(13, 4))
stations = ['A', 'B', 'C']
for i, name in enumerate(stations):
    axes[0].plot(history[:, i], label=f'Station {name}', marker='o', markersize=3)
axes[0].set_title('Bike Distribution Over Time')
axes[0].set_xlabel('Time step')
axes[0].set_ylabel('Proportion')
axes[0].legend()
axes[0].set_ylim(0, 1)

axes[1].bar(stations, history[-1], color=['#93c5fd', '#86efac', '#fde047'], edgecolor='black')
axes[1].axhline(stationary[0], color='blue', linestyle='--', alpha=0.5)
axes[1].set_title('Long-Term Distribution vs Stationary')
axes[1].set_ylabel('Proportion')
plt.tight_layout()
plt.show()

7. Markov Chain 案例：天气预测

场景：今天天气已知，预测明天、后天以及长期的天气概率。状态只有三种：晴天 Sunny、多云 Cloudy、雨天 Rainy。

天气转移模型

状态空间：S = {Sunny, Cloudy, Rainy}

转移矩阵示例：

SunnyCloudyRainy
Sunny0.70.20.1
Cloudy0.30.40.3
Rainy0.20.30.5

问题：

  • 已知今天多云，明天晴天的概率是多少？
  • 长期天气比例（稳态分布）是什么？
takeaway：Markov 链把“序列预测”变成矩阵乘法。 
import numpy as np
import matplotlib.pyplot as plt

# 天气转移矩阵
states = ['Sunny', 'Cloudy', 'Rainy']
P = np.array([
    [0.7, 0.2, 0.1],
    [0.3, 0.4, 0.3],
    [0.2, 0.3, 0.5]
])

# 今天多云 -> 明天概率
pi_today = np.array([0, 1, 0])
pi_tomorrow = pi_today @ P
print('Today cloudy -> tomorrow:', dict(zip(states, pi_tomorrow)))

# 长期稳态
steps = 50
history = [pi_today]
for _ in range(steps):
    history.append(history[-1] @ P)
history = np.array(history)

# 稳态解析
eigenvalues, eigenvectors = np.linalg.eig(P.T)
idx = np.argmin(np.abs(eigenvalues - 1))
stationary = eigenvectors[:, idx].real
stationary /= stationary.sum()
print('Stationary:', dict(zip(states, stationary)))

# 可视化
fig, axes = plt.subplots(1, 2, figsize=(13, 4))
for i, name in enumerate(states):
    axes[0].plot(history[:, i], label=name, marker='o', markersize=3)
axes[0].set_title('Weather Probability from Cloudy Today')
axes[0].set_xlabel('Days ahead')
axes[0].set_ylabel('Probability')
axes[0].legend()
axes[0].set_ylim(0, 1)

axes[1].bar(states, stationary, color=['#fde047', '#93c5fd', '#60a5fa'], edgecolor='black')
axes[1].set_title('Long-Run Weather Proportion')
axes[1].set_ylabel('Probability')
plt.tight_layout()
plt.show()

8. 三种随机建模视角对比

方法 核心思想 适用场景 输出
Monte Carlo 重复随机抽样，从大量试验中估计概率 风险分析、工期、落点、面积估计 结果分布、概率、置信区间
Agent-Based 个体简单规则 → 群体涌现行为 生态系统、人群、交通、商场 时序演化、相空间、涌现模式
Markov Chain 下一状态只依赖于当前状态 天气、共享单车、PageRank、NLP 状态概率演化、稳态分布
建模起手式：拿到带不确定性的题，先问： (1) 不确定变量是什么？适合什么分布？ (2) 个体之间有没有交互？（有 → ABS；没有 → Monte Carlo 或 Markov） (3) 未来状态是否只依赖当前？（是 → Markov） (4) 要估计的是概率/风险，还是长期稳定状态？
